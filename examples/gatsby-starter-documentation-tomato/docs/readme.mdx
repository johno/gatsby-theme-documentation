# Gatsby Documentation Theme

`gatsby-theme-documentation` is a minimalist Gatsby Theme for
docs sites built with [MDX](https://mdxjs.com) and
[Theme UI](https://theme-ui.com).

## Installation

```
yarn add gatsby-theme-documentation
```

## Usage

```js
// gatsby-config.js
module.exports = {
  plugins: [
    'gatsby-theme-documentation'
  ]
}
```

[See the official docs](https://github.com/johno/gatsby-theme-documentation)
